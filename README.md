# 🔢 SUDOKU SOLVER (GDSC Project)
2024 GDSC(Google Developer Student Clubs) 활동의 일환으로 개발된 스도쿠 솔버(Sudoku Solver) 웹 애플리케이션입니다.

사용자가 스도쿠 퍼즐을 입력하면 알고리즘을 통해 해답을 찾아주거나, 새로운 스도쿠 퍼즐을 만드는 기능을 제공합니다.

## 👨‍💻 참여자 (Contributors)
- 박재현(Frontend)
- 김민지(Backend)

## 🛠 기술 스택 (Tech Stack)
Frontend: JavaScript, HTML, CSS

Backend: Node.js

Algorithm: Backtracking (DFS 기반)
